# PostSkill Demo Case — AI醒觉社

> 一个最小可展示案例：从单个主题，生成多套文案、图片素材结果，并整理成可审核内容。

---

## 输入主题

`AI醒觉社`

## 执行命令

```bash
python postskill.py run --topic "AI醒觉社"
```

---

## 预期流水线

```text
输入主题
  ↓
生成多风格文案
  ↓
生成对应图片
  ↓
整理为 Markdown / 可协作素材
  ↓
接入后续发布动作
```

---

## 示例输出（图片阶段）

当前仓库通过 `scripts/image_generator.py` 调用 PonyFlash，为每套文案生成匹配配图。

### 图片策略
- 尺寸：768×1024（3:4）
- 分辨率：2K
- 按文案风格自动匹配 prompt

### 风格映射示例
- 干货型 → 专业、极简、蓝色科技风
- 故事型 → 温暖、治愈、带情绪氛围
- 金句型 → 强视觉、海报感、记忆点强

---

## 示例输出（素材文档）

执行完成后，会生成一份 Markdown 素材文档，用于：

- 审核
- 修改
- 二次编辑
- 对照查看图文是否匹配

## 这个 demo 说明了什么

这个案例最重要的不是“生成了几句文案”，而是：

**一个主题，已经可以被拉成一条可复用的内容生产流水线。**

这也是 PostSkill 最值得继续迭代的方向。
