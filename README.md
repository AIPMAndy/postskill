# PostSkill

端到端图文批量生产与自动发布工具。

## 核心能力

1. **文案生成** - 自动生成多种风格文案
2. **配图生成** - 调用PonyFlash生成配套图片
3. **飞书文档** - 自动创建图文对照文档
4. **自动发布** - 浏览器自动化多平台发布

## 快速开始

```bash
# 安装
codex skill install postskill

# 使用
postskill run --topic "AI醒觉社" --publish --platforms wechat,xiaohongshu
```
